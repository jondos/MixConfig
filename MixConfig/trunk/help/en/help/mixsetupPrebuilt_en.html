Using prebuilt binaries

If you're using either Debian GNU/Linux or Novell SuSE Linux as operating system for your Mix, we recommend you to use the prebuilt packages for your distribution. The advantage is that installation and updates will work much more convenient without any manual interaction in the file system and with a compiler. We provide binaries for other systems, too, but you will have to do some things on the console to get them running.
  • Debian GNU/Linux
  • Novell SuSE Linux
  • Other Linux/UNIX distribution

The Mix server precompiled binaries are just distributed for Linux / UNIX systems. For reasons of security and stability but also because it has not been developed and most tested on this platform we recommended very much this system for running a Mix.

If you want to run a Mix using a Windows system you need to compile it by your own. You will need either Eclipse or a Microsoft Visual C++ .NET For additional information please refer our developer manual at:
