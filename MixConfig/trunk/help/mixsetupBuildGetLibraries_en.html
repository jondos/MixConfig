Using a self-compiled Mix executable on Linux/UNIX

You will need several libraries to compile the Mix server code.

Installing the libraries

Precompiled binaries and sources of the needed libraries are available on these websites:
OpenSSL for cryptographic functions

Xerces-C++ for XML functions

If you even want to compile the libraries by your own, please follow the given links to the original websites of each library and check for documentation how to do this there. This documentation will give you just an explanation how to work with precompiled libraries.

The most convenient way to get the libraries installed on your system is to use the package manager of your distribution. Using, for example, SuSE 9.3 this is Yast2, using Debian 3.1 this is the text based and even more simple apt-get frontend. If you are not familiar with these tools please check your distribution's documentation.
