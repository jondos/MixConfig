The MixConfig tool

Since the configuration of a MIX server has become more and more difficult and too complex to be maintained by hand, we developed a configuration tool called MixConfig. It is both a tool for creating the XML file that is later used by your MIX server, and a certificate management tool for the server's cryptographic keys. MixConfig is a Java application which will run on all Java Runtime Environments since 1.1.8 and on Microsoft JView. You may download the tool here. Start it by typing the following command:
java -jar MixConfig.jar
If you use Microsoft Windows XP, this command will do the same:
jview /cp MixConfig.jar mixconfig.MixConfig
You get a list of possible options with
java -jar MixConfig.jar --help
or, on Windows XP, with
jview /cp MixConfig.jar mixconfig.MixConfig --help
