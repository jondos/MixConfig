Start Screen

The MixConfig tool has two modes of operation: The Wizard mode, where you get a complete guidance to create the configuration file for your mix, and the expert mode, where you get some more options, but also less help. (See also View menu for further details). If you choose one of the following options, the tool will start in the wizard mode:
  • Create new configuration
    Creates a new configuration with default values.
  • Load/Resume existing configuration
    Loads a configuration from a file.
  • Configure Cascade for existing configuration
    Load a configuration from a file, and jumps directly to the "Configure Cascade" dialog. This does only work if the mix is dynamic and the last one in the cascade. (Currently not available)
