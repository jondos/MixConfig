File menu

The File menu contains options for creating, loading and saving configuration files in the XML format. The different options are in detail:
  • New configuration (expert mode)
    Create a new configuration containing initial default values. To prevent loss of data, please make sure to save your existing configuration before creating a new one. Note that the newly created configuration will be shown in expert mode.
  • Open (expert mode)
    Load a configuration from a file. The configuration is shown in expert mode.
  • Open from clipboard (expert mode)
    Load a configuration from the clipboard. The configuration is shown in expert mode.
  • Check
    Check if all of the fields in the current configuration are consistent and shows error messages if not. If you try to run a Mix with an inconsistent configuration it will not work or at least not in the way it should.
  • Save
    Save the configuration to a file. You can save a configuration whenever you want while you will be warned if there are any inconsistencies.
  • Save as
    Save the configuration to a file that needs to be specified. You may save your configuration whenever you want while you will be warned if there are any inconsistencies.
  • Save to clipboard
    Save the configuration to the clipboard. You may save a configuration whenever you want while you will be warned if there are any inconsistencies.
  • Exit
    Close the program. Please save your configuration before exiting!
