Using prebuilt binaries

If you have installed SuSE Linux on your Mix server, you may use the precompiled installation packages we provide.

SuSE precompiled installation packages

Insert the URL corresponding to your system in yast's installations sources:
  • http://anon.inf.tu-dresden.de/develop/rpm/SUSE-9.0
  • http://anon.inf.tu-dresden.de/develop/rpm/SUSE-9.1
  • http://anon.inf.tu-dresden.de/develop/rpm/SUSE-9.2
  • http://anon.inf.tu-dresden.de/develop/rpm/SUSE-9.3
Now you can install the AN.ON mix with yast by searching for "Mix". Copy the configuration file(s) created by the mix configuration tool in the directory /etc/mix and start the mix by typing sudo /etc/init.d/mix start. If you deploy more than one configuration file, a separate mix will be started for each of the files. To stop all running mixes, type sudo /etc/init.d/mix stop.
