File menu

This menu contains commands for creating, loading and saving configurations.
  • New configuration (expert mode)
    Creates a new configuration with default values. Please make sure to save your existing configuration before. Note that this configuration is opened in expert mode.
  • Open (expert mode)
    Loads a configuration from a file. Note that this configuration is opened in expert mode.
  • Open from clipboard (expert mode)
    Loads a configuration from the text stored in the clipboard. Note that this configuration is opened in expert mode.
  • Check configuration consistency
    Checks if all fields in the current configuration are consistent and gives error messages if not. If you try to run a mix with an inconsitent configuration it won't work or at least not as it should.
  • Save
    Saves the configuration to a file. You may save your configuration whenever you want and you will be warned if it is inconsistent.
  • Save as
    Save the configuration a file that must be specified. You may save your configuration whenever you want and you will be warned if it is inconsistent.
  • Save to clipboard
    Save the configuration to the clipboard. You may save your configuration whenever you want and you will be warned if it is inconsistent.
  • Exit
    Closes the program. Please save your configuration before doing this!
