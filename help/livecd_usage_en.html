Live-CD (MixOnCD)

Booting

The Live-CD should boot automatically when you turn on your PC. If booting fails, choose one of the different debug options from the start menu. You can now switch to the logging console by typing CTRL+ALT+F2 to see if the mix has started properly. You can switch to other consoles via CTRL+ALT+Fx.

Local login

Once your mix has started, a login dialog is presented. You may now type the username and password of the user you wish to log in. If you did not specify any password in the MixOnCD configuration, just type root. If you did not specify any password in the MixOnCD configuration, you may not login to your mix via network, but you can login with your local keyboard. just type root in this case.

  • Two-session CD with configuration file in second session
    Create a multi-session disc with the iso-file in the first session and put the configuration file created by this tool on the second session.
  • Single-session CD with configuration file on a USB device
    Copy your configuration to a USB device, for example a USB stick. Insert the Live-CD, switch off your computer, plug the device into any USB port, and switch power on again.
Your mix will now start automatically, but it may take some minutes to boot.
